GlobeTrotter is a web-based travel planning application developed as part of a hackathon.

The idea behind this project is to make trip planning easier by allowing users to
create trips, add multiple cities, and organize basic itinerary details in one place.

## Current Progress
- Repository and project structure created
- Initial UI planning for core pages
- Basic Create Trip screen started

## Technologies Used
- HTML
- CSS
- JavaScript

## Planned Functionality
- User login and signup
- Create and manage trips
- Add cities and activities to a trip
- View itinerary in a simple timeline or list
- Basic budget estimation

## Project Status
This project is still under development and was built within the limited time of the hackathon.
